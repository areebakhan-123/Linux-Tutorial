Linux Basic Tutorial

1. View Current Directory

Command: pwd

Displays the path of your current working directory.

2. List Files and Folders

Command: ls

Lists all files and directories in the current directory.

3. Change Directory

Command: cd foldername

Moves you into a different folder (directory).

4. Create a New File

Commands: touch filename.txt

Creates a new empty file.

5. Create a New Folder

Command: mkdir foldername

Creates a new directory (folder).

6. Delete a File

Command: rm filename.txt

Deletes a file from the directory.

7. Delete a Folder

Command: rm -r foldername

Deletes a folder and all its contents.

8. View File Contents

Command: cat filename.txt or less filename.txt

Displays the contents of a file.

9. Copy Files

Command: cp source.txt destination.txt

Copies a file to another location or with a new name.

10. Move or Rename Files

Command: mv source.txt destination.txt

Moves or renames a file.

11. Install New Software

Command: sudo apt install packagename

Installs new software packages (for Ubuntu).
